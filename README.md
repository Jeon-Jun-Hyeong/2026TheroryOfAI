# Auto Data Insight

브라우저에서 로컬 파일을 바로 분석하는 정적 웹사이트입니다.

## 기능

- `csv`, `xlsx`, `json`, `xml` 업로드 지원
- 업로드 후 자동 전처리
- 숫자형 / 날짜형 / 범주형 컬럼 추론
- 자동 차트 추천
- 텍스트 기반 분석 요약
- 예시 데이터 포맷 생성 및 다운로드

## 실행

`index.html`을 브라우저에서 열면 됩니다.

외부 CDN을 사용합니다.

- `SheetJS` for CSV/XLSX parsing and XLSX export
- `Plotly` for chart rendering
